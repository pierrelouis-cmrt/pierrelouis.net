Colophon

A peek behind the making of this portfolio.

Build & Stack

Designed and coded by me with semantic HTML, modern CSS, and vanilla JavaScript. I sprinkle in Tailwind and AlpineJS for a few interactive details (like the command palette). The entire codebase is open-source on GitHub under the MIT license.

I design, prototype, and test directly in code using VS Code.

Type & Icons

The interface leans on the system UI font stack for performance and clarity, with a touch of Libre Baskerville for accents. Icons are from Lucide, a free and open-source set built by a great community.

Privacy-Friendly Analytics

I use Umami, a lightweight, open-source, cookieless analytics tool. I simply want to understand how the site is used (how you discover it, which pages you visit, etc.). That's it. No ads, no cross-site tracking, no cookies, and no personally identifiable data — ever.

Sustainability

Performance and footprint matter. This site is cleaner than 96% of tested web pages. Each visit produces about 0.02g of CO₂. It runs on verified green hosting, and it's featured on lowwwcarbon with an A+ rating.

What helps keep it lean: minimal JavaScript, compressed assets, system fonts by default, carefully budgeted images, and a straightforward architecture.
